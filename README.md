# Stone-Paper-and-Scissors
Stone, Paper, and Scissors is one of the most played games during our childhood. This is my 2nd mini project which has been made with the help of HTML, CSS, and JavaScript. It focuses on DOM manipulation, event handling, game logic, and responsive design. Unlike the traditional version, it consists of a Computer Opponent, with whom you will compete!
